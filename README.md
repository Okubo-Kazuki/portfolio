# ポートフォリオ

このリポジトリは、React + Vite + Tailwind CSS で構築したポートフォリオサイトのソースコードです。

## 概要

- フロントエンド開発に特化したポートフォリオ
- React 19 + TypeScript で実装
- Vite をビルドツールとして使用
- Tailwind CSS でレスポンシブデザインを記述
- 画像やプロジェクトカードを動的に表示

## 主な技術

- React
- TypeScript
- Vite
- Tailwind CSS
- ESLint

## ファイル構成

- `src/` - アプリケーション本体
  - `components/` - UI コンポーネント
  - `assets/img/` - 画像アセット
  - `main.tsx` - エントリーポイント
  - `App.tsx` - ルートコンポーネント
- `public/` - 静的ファイル
- `vite.config.ts` - Vite 設定
- `tsconfig.json` - TypeScript 設定
